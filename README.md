# 🎬 Netflix Analytics Dashboard

An interactive **Netflix Movies and TV Shows Analysis Dashboard** built using **Python, Plotly, and Streamlit**. This project performs data preprocessing, exploratory data analysis (EDA), and interactive visualization to uncover meaningful insights from the Netflix Movies and TV Shows dataset.

---

# 📌 Project Overview

The Netflix Analytics Dashboard is a university-level Data Analytics project that analyzes Netflix's content library using Python. The project focuses on transforming raw data into meaningful insights through data cleaning, preprocessing, visualization, and dashboard development.

The dashboard allows users to explore Netflix content interactively using filters and Plotly visualizations.

---

# 🎯 Objectives

- Analyze Netflix Movies and TV Shows dataset.
- Perform Data Cleaning and Preprocessing.
- Explore content distribution.
- Analyze release year trends.
- Study content ratings.
- Identify top producing countries.
- Discover popular genres.
- Analyze movie durations.
- Identify top directors and actors.
- Develop an interactive Streamlit dashboard.
- Generate business insights using data visualization.

---

# 💡 Key Business Insights

- Movies dominate Netflix's content library.
- Netflix significantly expanded after 2015.
- TV-MA is the most common content rating.
- The United States contributes the highest number of titles.
- International Movies are among the most popular genres.
- Most content is added within 0–3 years of release.
- Netflix frequently collaborates with popular directors and actors.
- Most movies have durations between 90 and 120 minutes.
